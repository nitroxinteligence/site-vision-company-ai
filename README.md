# Vision AI

Landing page institucional da Vision AI - empresa especializada em agentes de inteligência artificial para automação de processos empresariais.

## Sobre

A Vision AI oferece soluções de IA personalizadas para empresas que buscam escalar operações, reduzir custos e automatizar processos repetitivos. Nossos agentes inteligentes atuam em áreas como SDR, Customer Success, Vendas, Financeiro e Gestão de Dados.

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS v4
- GSAP (animações)
- TypeScript

## Desenvolvimento

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000)
